Frequently Asked Questions

How do I create a new task?

Go to the "Create Task" page, fill in the task details, and click save. You’ll see the new task appear on your dashboard.

Can I edit or delete my tasks?

Yes! You can edit or delete any task from the “My Tasks” page. Use the edit or delete buttons next to each task.

Why can't I see my tasks after logging in?

Make sure you’re logged in with the same account you used to create them. Each user’s tasks are private and secure.

How can I enable email reminders?

You can enable or disable email notifications from your account settings page (coming soon in future release).

Is there a mobile version?

Yes, the web application is fully responsive and works on all mobile browsers. A dedicated mobile app is coming soon.
